### Prerequisites

This is an example of how to list things you need to use the software and how to install them.

* cd
  ```sh
  cd frontend
  ```

* yarn
  ```sh
  yarn install
  ```

* run
  ```sh
  yarn dev
  ```
  
---------------------------------------

* cd
  ```sh
  cd backend
  ```

* venv
  ```sh
  python venv venv
  ```

* activate
  ```sh
  venv/Scripts/activate
  ```

* pip
  ```sh
  pip install -r requirements.txt

  ```

* run
  ```sh
  python manage.py runserver

  ```
